I created this package for the purpose of completing the workout 3 assignment for Stat 133 at UC Berkeley. The following binomial package contains functions, objects and methods regarding binomial distributions.

Basic Statistics Functions

The most basic functions in this package are those to calculate different statistics on the binomial distribution of the userâ€™s choice. The user will be able to calculate the mean, variance, skewness, and more of the binomial distribution.

number_trials <- 5
probability_success <- 0.5

Mean

bin_mean(number_trials,probability_success)
## [1] 2.5

Variance

bin_variance(number_trials,probability_success)
## [1] 1.25

Mode

bin_mode(number_trials,probability_success)
## [1] 3

Skewness

bin_skewness(number_trials,probability_success)
## [1] 0

Kurtosis

bin_kurtosis(number_trials, probability_success)
## [1] -0.4

All of these measures have the following constraints: - probability must be a number between 0 and 1 - number of trials must be non-negative

Other Distribution Objects

This package also has the option of creating the following distribution objects: - bindis - bincum Both of them inherit from the object data.frame.

Binomial Distribution Objects

The â€˜bindisâ€™ objects are displayed as a data.frame with columns for number of successes and probability that we will achieve that numbe of successes.

Cumulative Binomial Distribution Objects

The â€˜bincumâ€™ objects are also displayed as a data.frame, much like â€˜bindisâ€™, except they also include a column showing the cumulative probability of each success.
